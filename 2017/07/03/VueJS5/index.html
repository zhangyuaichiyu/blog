即时搜索

<div id="main" v-cloak>
        <div class="bar">
            <input type="text" v-model="searchString" placeholder="搜索">
        </div>
        <ul>
            <li v-for="i in articles | searchFor searchString">
                <a v-bind:href="i.url">
                    <img v-bind:src="i.image" alt="">
                </a>
                    <p>{{i.title}}</p>
            </li>
        </ul>
    </div>

这个例子主要应用了vue.js的自定义过滤器，可以通过Vue.filter()注册一个全局过滤器，具体用法可以参考这里，vue.js也提供了一些内置过滤器。
