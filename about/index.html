关于我

我虽然是一名女生，单对计算机有特别的爱好，不想忙碌于重复无聊的工作，也不喜欢每天做一些没什么价值的事，我认为最有价值的行业就是计算机类高科技，它可以改变人们的生活方式并促进社会的发展、文明的进步。

自由就对计算机黑技术有好感，单那时认知有限，就只能羡慕那些游走于计算机世界的黑客们啦，在电视剧中看到破案人员在电脑上输了几个命令，就能出现一些比较炫的英文，虽然看不懂，但是他们能进入各种常人不能进入的系统。

就现在而言我目前在一家软件转运营的公司以一个前端工程师的身份实习，单现在我也面临毕业再就业问题，每想到此就头大，走一步说一步吧，人生就是这样有着压力才有意思。

所涉及的前端领域

目前我所擅长的是前端技术，所学和应用的知识有:

  • 前端框架的搭建
  • html、html5
  • css3
  • 微信端页面开发
  • 微信小程序
  • 微信订阅号和服务号
  • Javascript
  • Jquery
  • Jquery插件的编写
  • JqueryMouble
  • AngularJs
  • AngularJs2
  • node JS

对常用的一些插件也用的顺手如:

  • 表单验证
  • 头像上传
  • 百度图片上传
  • echart的图表等

但毕竟我是学后台java的，对此我在公司里也要了些接口联调的任务用来温习java不至于忘记，现在就是业余时间学习java后台的知识啦。目前用的工具是简单的xampp
